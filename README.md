# AI Learning Recommendation System

## Project Description
An AI-powered learning recommendation system that provides:
- Subject-wise quizzes
- Performance analysis
- Weak topic detection
- Recommended learning videos
- Student progress tracking

## Features
- Python & DBMS quizzes
- Randomized questions
- Timer-based quiz
- Weak topic analysis
- Chart-based performance dashboard
- Video recommendations

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- Chart.js

## How to Run

```bash
pip install -r requirements.txt
python app.py
```
